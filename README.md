# mp3-renamer
Rename mp3 from metadata

## Description
Find mp3 files recursively and rename them as "Track Number - Song Tittle"

## Features
- Finds mp3 with low Kbps
- Finds mp3 with no metada
- Finds non-mp3 files
- Generates log file

## Libraries
eye3D

## Usage
```bash
python main.py musicPath
```
